# TraceLink

Превращает длинные ссылки в короткие — и показывает, сколько раз по каждой
короткой ссылке кликнули, с какого устройства и браузера, в какие дни.

В комплекте — аккуратная админка: войдите, вставьте длинный URL, получите
короткую ссылку с готовым **QR-кодом** и наблюдайте за кликами на живом
дашборде с графиками.

<p align="center"><em>Шлюз · Админка · Сервис редиректов · Сервис аналитики · PostgreSQL · Redis — всё запускается одной командой.</em></p>

---

## Что умеет

- **Сокращает** любую ссылку, с необязательным сроком жизни (TTL) и флагом «приватная».
- **Перенаправляет** мгновенно — популярные ссылки отдаются из кэша Redis.
- **QR-код** генерируется локально прямо на дашборде (без сторонних сервисов).
- **Считает** каждый клик: всего, по дням, разбивка по устройствам и браузерам.
- **Тёмная тема**, обновление статистики в реальном времени, панель здоровья
  системы и просмотр логов с трассировкой запросов.

## Запуск одной командой

Нужен только [Docker](https://docs.docker.com/get-docker/).

```bash
cp .env.example .env        # 1. создать файл настроек
docker compose up -d        # 2. запустить всё
```

Откройте **<http://localhost:8080>**.

> Первый запуск ещё и собирает образы — подождите минуту. Дальше `up` мгновенный.

### Вход в админку

| | |
|---|---|
| **Адрес** | <http://localhost:8080> |
| **Логин** | `admin` |
| **Пароль** | `admin_dev_2026` |

> Это **локальные значения по умолчанию** из `.env`. Перед использованием
> где-либо всерьёз поменяйте `ADMIN_PASSWORD` и остальные секреты — см.
> [`docs/SECURITY.md`](docs/SECURITY.md).

### Остановить

```bash
docker compose down         # остановить
docker compose down -v      # …и стереть базу данных
```

## Попробовать за 20 секунд

1. Войдите с данными выше.
2. Вставьте любой длинный URL и нажмите **Shorten**.
3. Появится карточка с короткой ссылкой и QR-кодом.
4. Откройте короткую ссылку в новой вкладке — вас перенаправит на оригинал.
5. Обновлять дашборд не нужно — клик сам появится на графиках (поллинг каждые 5 c).

## Что внутри

Пять контейнеров, запускаемых вместе через Docker Compose:

| Компонент | Назначение |
|---|---|
| **nginx** | Единая точка входа — направляет на админку или выполняет редирект |
| **admin-panel** | Сайт, в который вы входите (ссылки, QR, графики, логи) |
| **redirect-service** | Создаёт короткие коды и делает быстрые редиректы |
| **stats-service** | Собирает клики и считает статистику |
| **PostgreSQL + Redis** | Хранят данные и кэшируют горячие ссылки |
